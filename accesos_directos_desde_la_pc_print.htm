12. Accesos directos desde la PC

 
Las transacciones que se utilizan con frecuencia se pueden crear como los llamados accesos rápidos. Esto sólo es posible si utiliza un PC con el sistema operativo Windows.
 
  • Posicione el cursor sobre una transacción del árbol de menú. A  continuación, seleccione la opción de menú Tratar -> Crear acceso  rápido. En el desktop de su ordenador se creará un acceso rápido para la correspondiente transacción.
 
 
  • Finalmente puede iniciar la transacción haciendo doble clic sobre el acceso rápido. Además, puede arrastrar el acceso rápido a una ventana actual del Sistema R/3, para iniciar en ella la transacción del  acceso rápido.
     
     
  • Tenga en cuenta que en esta posición sólo se pueden crear accesos  rápidos para las transacciones, mientras que para los reportes y demás objetos que se encuentran en los árboles de menú no es posible. Si desea crear accesos rápidos para reports utilice los iconos correspondientes para crear accesos rápidos en SAP SAPGui.
