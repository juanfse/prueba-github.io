4. ¿Cómo ingresar o “loguearse”?

 
1.Haga click en el ícono SAP Logon, en su escritorio
2. Coloque su usuario y contraseña/password
 

MANDATE

Este campo permite seleccionar el mandante al cual se quiere acceder. El mandante es una unidad organizativa dentro del sistema SAP, identificada por un número único, que contiene datos específicos como configuración y transacciones
Arriba

USUARIOS

Este es el campo donde se debe ingresar el nombre de usuario asignado para acceder al sistema SAP. Cada usuario tiene un perfil que determina sus permisos y accesos dentro del sistema.
Arriba

CONTRASEÑA

Los puntos de la contraseña están permanentemente ahí, ingrese su contraseña
Arriba

IDIOMA DEL SISTEMA

Aquí se especifica el idioma en el cual se desea utilizar la interfaz de SAP. En la imagen, el idioma seleccionado es "ES" (Español).
Arriba
 
3.La primera vez que ingrese al sistema le pedirá que cree una contraseña nueva

BOTON DE CONFIRMACION

       Dé click en el check verde para continuar
 
Arriba
La contraseña debe cumplir con estas características:
• Debe contener como mínimo una letra mayúscula
• Debe contener mínimo una letra minúscula.
• Debe contener mínimo un dígito.
• Debe contener como mínimo un carácter especial.
• El sistema pedirá cambio de clave de acceso cada 30 días.
• Solo puede volver a usar una clave anterior si ha cambiado 5 veces de clave.
