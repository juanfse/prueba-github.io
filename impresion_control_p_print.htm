10.2. Impresion Control+P

Al presionar Ctrl + P, SAP abre la ventana de impresión donde puedes seleccionar la impresora, configurar las opciones de impresión (como el número de copias o el rango de páginas a imprimir), y finalmente enviar el documento a la impresora seleccionada. Esta función es útil para generar copias físicas de documentos importantes como órdenes de compra, facturas, informes financieros, entre otros, que son generados dentro del sistema SAP.
 
1.Tocar control+P , se despliega la Ventana de seleccion de mpresora, se puede utilizar el dispositivo “PDF1”
2. Seleccionar la impresora que se desea utilizar o el formato PDF
